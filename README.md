通过对python的学习，我还学会了制作简易游戏。
```python
import random
def caishu():
    h = random.randint(1,10)

    a = 5
    while a > 0:
        s = int(input("数字是啥"))
        if s == h:
            print("牛")
            break
        if s > h :
            print("大了")
        else:
            print("小了")
        a = a - 1
    else:
        print("gameover")
caishu()
```
